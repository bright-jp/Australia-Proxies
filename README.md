## FAQ

### オーストラリアプロキシサーバーとは何ですか？
オーストラリアプロキシサーバーとは、オーストラリアに設置されたサーバーであり、お客様のデバイスとインターネットの間に入る仲介役として機能します。匿名性を提供し、制限やブロックの回避に役立ち、Webコンテンツのスクレイピングを可能にし、コンテンツフィルタリングポリシーを適用します。

### オーストラリア全域でZIPコードをターゲットにできますか？
はい、オーストラリアのZIPコードレベルのターゲティングを使用してIPを選択できます。Bright Dataでは、米国プロキシの都市および州レベルのターゲティングも提供しています。

### 利用できるプランの種類は何ですか？ 
Bright Dataは、以下を含む柔軟な料金モデルを提供しています。

- **Pay-As-You-Go**: GBあたりの固定料金です。
- **サブスクリプションプラン**: 月次、年次、カスタムオプションです。

### Bright Dataのオーストラリアプロキシは準拠しており、安全に使用できますか？
Bright Dataのプロキシは倫理的に取得されており、GDPRおよびCCPAを含む関連するすべてのデータ保護法に準拠しています。

### 専任のサポートは利用できますか？
専任サポートチームが24時間365日、お客様を支援します。お問い合わせいただき、お客様のニーズについてご相談ください。Dedicated proxy networkのメリットを最大限にご活用いただけます。
